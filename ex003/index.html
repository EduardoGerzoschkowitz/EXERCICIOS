A tecla exclamação ! cria o código base em HTML de um site, onde existe a cabeça e o corpo. Para adicionar texto use h1

O comando & ( famoso e comercial ) abre a biblioteca de atalhos para simbolos, como por exemplo: ©(símbolo copyright) €(símbolo do euro)

Para adicionar emojis use &#x e o código do emoji encontrado no Google. Exemplo: 🖖 🤓
