Resumo

Entender a lógica é mais importante do que decorar os comandos.
HTML é para o texto(Hypertext march language),imagens,vídeos e tabelas do site.
CSS e para o design de cores, sombras, tamanhos e posicionamento.
Javascript é para interações como menus, pop-ups e animações.
Desenvolvedor Front-end(client-side) e quem usa html,css e javascript
Back-end(server-side) é o que opera o servidor em PHP.
Fullstack é o que faz os dois. Programador de pynthon, java... desenvolvedor html,css..
